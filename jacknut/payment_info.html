personal checks

If you would like to order some gear and pay with a personal check, you can simply write up the order with the items you would like. Make sure to include style (mens/womens), color, design and any other pertinent information. Total the prices and send a check for the total. Stickers and magnets have shipping costs included in the price. For shirts, and $5 shipping and handling per shirt. Send the order, the mailing address where we should send the items, a valid email address (in case there is a problem with the order), and a check made out to 'jon rogers' to the following address:

jacknut
